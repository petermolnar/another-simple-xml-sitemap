== Description ==

A very simple dynamic sitemap plugin, to generate XML sitemaps.

It only includes singular (posts, pages, etc. ) permalinks, so no taxonomy (tag, archive, category, etc.) links will be included at all.

This plugin writes the sitemap url in your robots.txt file, which is a good for search engines.

= Credit =

This plugin is a fork of thank you for [Another Simple XML Sitemap](https://wordpress.org/plugins/another-simple-xml-sitemap/) because the original plugin was discontinued.


== Installation ==


1. Upload contents of `another-simple-xml-sitemap.zip` to the `/wp-content/plugins/` directory
2. Activate the plugin through the `Plugins` menu in WordPress
